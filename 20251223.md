question 1:
    shell里面的命令行后加入'参数是什么意思
在Shell命令行中，单引号（'）用于创建一个字面字符串，会原样保留引号内的所有字符，不进行任何特殊解释或扩展。

例如：
echo "
> helloworld"
则会输出helloworld 

'也如此

question 2:
    ubuntu无法使用键盘上的亮度控制按键来调整亮度
1. 执行下面这个命令，检查亮度控制文件是否存在
ls /sys/class/backlight/

2. 后安装系统控制亮度的工具brightnessctl
sudo apt install brightnessctl

3. 设置亮度，reboot系统之后亮度设置完毕
brightnessctl set X%
