question 2:
    getpwuid()和getgrgid()的返回值

成功：返回值为一个指向struct passwd | struct gr 的指针
失败：NULL，设置errno

重要注意事项：
1.返回值指向静态内存：两个函数返回的指针都指向静态内存区域，后续调用会覆盖这些内容。如果需要保存信息，应该复制结构体内容：

```
struct passwd pwd_copy = *getpwuid(uid); //防止覆盖原本内容
```

2.线程安全版本：大多数系统提供线程安全版本：
```
int getpwuid_r(uid_t uid, struct passwd *pwd,
               char *buf, size_t buflen, struct passwd **result);

int getgrgid_r(gid_t gid, struct group *grp,
               char *buf, size_t buflen, struct group **result);
```
