question 1:
    如何安装 nvidia 驱动

一般默认 ubuntu 的驱动为 Nouveou 驱动，开源但性能差。
安装电脑附加的驱动，如nvidia-driver-580-open ,运行下面的指令
sudo apt-get update
sudo apt install nvidia-driver-580-open (--fix-missing)

如果无法安装，更换一个镜像源，或者加入 () 的参数。

安装完成之后 reboot ,用 nvidia-smi 查看是否加载。
也可以调节电脑屏幕亮度来验证。

nvidia-driver-580-open 使用体验：
响应速度快，操作流畅无前摇。
